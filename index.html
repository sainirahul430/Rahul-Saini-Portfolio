Agent Performance Analytics — BigQuery Pipeline

Built an automated analytics system to track and optimise sales agent performance using real-time call data, replacing manual reporting with scalable data pipelines.

45K+
Records/month
3 hrs
Saved daily
60
Agents analysed

Problem

Agent performance tracking was manual and delayed, limiting visibility into efficiency and preventing timely interventions.

Approach

Built ETL pipelines pulling data from 4 dialer APIs into BigQuery using Python and Apps Script with 3-hour refresh cycles.

Result

Enabled real-time monitoring and benchmarking, allowing managers to identify underperforming agents and take immediate action.

KEY INSIGHTS

  • Agents with high idle time showed 20–25% lower conversion rates
  • Connectivity % strongly correlated with revenue generation
  • Top-performing agents followed consistent call distribution patterns

TECH STACK

Python, SQL (window functions), BigQuery, Google Apps Script, Looker Studio

SAMPLE QUERY

SELECT agent_id,
       COUNT(*) AS total_calls,
       SUM(talk_time) AS total_talk_time,
       RANK() OVER (ORDER BY SUM(talk_time) DESC) AS performance_rank
FROM calls
GROUP BY agent_id;
